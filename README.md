# Terminal-Based RPG 🗡️

A lightweight, text-based role-playing game engineered from scratch in C++ to solidify object-oriented programming principles and low-level memory management. 

## Features
* **Dynamic Player State:** Tracks health, inventory, and experience progression.
* **Turn-Based Combat:** Implements a core gameplay loop with tactical decision-making.
* **Algorithmic Enemy AI:** Basic state-machine behavior for enemy encounters.

## Tech Stack
* **Language:** C++
* **Environment:** Linux Command Line
* **Version Control:** Git

## How to Run
(`g++ main.cpp Character.cpp Player.cpp Enemy.cpp -o rpg` then `./rpg`)
